# EduTrack – Student Management System 🎓

A responsive and interactive Student Management System built using **HTML, CSS, and JavaScript**.  
This project represents my frontend development journey and understanding of UI design, DOM manipulation, and basic data handling in JavaScript.

---

## 📌 Features

- Add new student records
- View student list in dashboard
- Update student information
- Delete student records
- Real-time UI updates using JavaScript
- Responsive design for different screen sizes

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 Purpose of Project

This project was built as a part of my frontend learning journey to practice:
- DOM manipulation
- Event handling
- UI design principles
- Structuring multi-page web applications
